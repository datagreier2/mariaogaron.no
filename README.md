# Maria & Aron Wedding Site

A simple React + Vite site for the wedding of Maria & Aron with Norwegian and English language support.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This site deploys to GitHub Pages via GitHub Actions on every push to `main`.
The custom domain is configured in `public/CNAME`.

## Languages

The site auto-detects browser language (Norwegian/English) and allows manual switching.
